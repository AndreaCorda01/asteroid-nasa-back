## Description

Backend API for NASA Asteroid endpoint that collect all event of asteroids near to earth.

## Installation

```bash
$ npm install
$ npm run dev
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
